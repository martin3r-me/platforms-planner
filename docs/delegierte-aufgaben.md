# Delegierte Aufgaben

Die Ansicht **Delegierte Aufgaben** zeigt alle Aufgaben, die du erstellt aber einer anderen Person als Verantwortlichem zugewiesen hast.

## Wann erscheint eine Aufgabe hier?

Eine Aufgabe erscheint in dieser Ansicht, wenn:

- **Du** der Ersteller bist
- Eine **andere Person** als Verantwortlicher eingetragen ist

So behältst du den Überblick über Arbeit, die du abgegeben hast.

## Aufbau

Die Struktur ist ähnlich wie bei "Meine Aufgaben":

- **Fällige Aufgaben** — Delegierte Aufgaben, die überfällig sind
- **Eingang** — Frisch delegierte Aufgaben
- **Gruppen** — Eigene Gruppen zur Organisation delegierter Arbeit
- **Erledigt** — Vom Verantwortlichen abgeschlossene Aufgaben

## Typischer Workflow

1. Erstelle eine Aufgabe (in einem Projekt oder persönlich)
2. Weise einen Verantwortlichen zu
3. Die Aufgabe erscheint bei dir unter "Delegierte Aufgaben" und beim Verantwortlichen unter "Meine Aufgaben"
4. Wenn der Verantwortliche die Aufgabe erledigt, siehst du das hier
