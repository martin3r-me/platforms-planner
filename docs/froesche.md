# Frösche — Eat the Frog

Die **Frösche-Ansicht** basiert auf der Produktivitätsmethode "Eat the Frog": Erledige die schwierigste oder unangenehmste Aufgabe zuerst.

## Was ist ein Frosch?

Ein Frosch ist eine Aufgabe, die besondere Aufmerksamkeit braucht — entweder weil du sie selbst als wichtig markiert hast, oder weil das System erkannt hat, dass sie zu oft aufgeschoben wurde.

## Zwei Arten von Fröschen

### Manuelle Frösche

Du kannst jede Aufgabe manuell als Frosch markieren, indem du in der Aufgabe den Frosch-Schalter aktivierst. So markierst du Aufgaben, die du bewusst priorisieren willst.

### Zwangs-Frösche

Das System erkennt automatisch Aufgaben, die zu oft verschoben wurden, und macht sie zu **Zwangs-Fröschen**. Eine Aufgabe wird automatisch zum Frosch, wenn:

- Das Fälligkeitsdatum **4 Mal oder öfter** verschoben wurde
- Oder: Das Fälligkeitsdatum **2 Mal oder öfter** verschoben wurde **und** die Aufgabe groß ist (13+ Story Points)

Zwangs-Frösche können **nicht manuell entfernt** werden — sie bleiben Frösche, bis sie erledigt sind.

## Die Frösche-Ansicht

Die Ansicht zeigt alle offenen Frösche, gruppiert nach Projekt. Du siehst auf einen Blick:

- Gesamtzahl der Frösche
- Davon Zwangs-Frösche
- Summe der Story Points

## Filter

- **Person** — Zeige Frösche eines bestimmten Teammitglieds
- **Projekt** — Filtere nach Projekt

## Tipps

- Schau morgens zuerst in die Frösche-Ansicht und erledige mindestens einen Frosch pro Tag
- Wenn eine Aufgabe zum Zwangs-Frosch wird, ist das ein Signal: Diese Aufgabe blockiert deinen Fortschritt
- Große Frösche lassen sich oft in kleinere Aufgaben aufteilen
